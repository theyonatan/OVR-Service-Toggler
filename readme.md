# OVR Service Toggle

A tiny tool that toggles the Oculus VR service (OVRService) on launch.

# Why?

Sometimes VR interferes with non-VR games (like Roblox).

- Want to play normally? → run the tool → VR turns off

- Want to play in VR? → run it again → VR turns on


# **Windows only**

Run it (will prompt you for admin) and it will detect and do everything alone.
